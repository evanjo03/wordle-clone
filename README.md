# Lit Wordle

Wordle clone built with Lit and web components.

## Usage

```
npm i lit-wordle
```

Import the side effect registration of `<lit-wordle>` via:

```
import 'lit-wordle/define.js';
```

Import the base class:

```
import { LitWordle } from 'lit-wordle';
```
